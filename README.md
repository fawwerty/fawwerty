## 🚀 Featured Projects

### 🛡️ CyberShield — AI Cyber Threat Detection System
> Real-time intrusion detection powered by an intelligent ML ensemble system

[![Live App](https://img.shields.io/badge/🔗%20Live%20App-cyber--shield--ai--cyber--threat--detect.vercel.app-1A56DB?style=for-the-badge&logo=vercel&logoColor=white)](https://cyber-shield-ai-cyber-threat-detect.vercel.app)

**Key Highlights:**

- ⚡ Built a multi-model ensemble system:
  - **Isolation Forest** — unsupervised anomaly detection
  - **Extra Trees** — supervised learning
  - **Deep Autoencoder** — neural anomaly detection (HuggingFace)
- 📊 Processes **20+ engineered network traffic features** with StandardScaler preprocessing
- 🚀 **FastAPI** backend serving real-time predictions
- 📱 Integrated with **React Native (Expo)** cross-platform mobile app
- ☁️ Models versioned and deployed via **HuggingFace Hub**

`Python` `FastAPI` `scikit-learn` `HuggingFace` `React Native` `Expo`
